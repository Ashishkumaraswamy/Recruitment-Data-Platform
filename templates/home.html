URL Suffix Functions
/divisions Get all divisions in JSON format
/divisions/<dcode> Get details about the division with the given dcode
/divisions/add Add a division
/divisions/update/<dcode> Update details of the division with the given dcode
/divisions/delete/<dcode> Delete details of the division with the given dcode
/users Get all users
/users/register Register a user to the Recruitment Data Platform
/users/login Login as a User into the Recruitment Data Platform
/users/update/<username> Update details of the user with the given username
/users/delete/<username> Delete details of the user with the given username
/jobs Get all jobs
/jobs/<JobId> Get details about the job with the given jobId
/jobs/add Add a new job
/jobs/update/<JobId> Update details of the job with the given jobId
/jobs/delete/<JobId> Delete job with the given jobId
